L'applicazione funziona tramite una barra di ricerca, simile a quella di Google, che permette agli utenti di cercare libri selezionando una categoria specifica (ad esempio, "Fantasia"). Una volta inserita la categoria, l'utente avvia la ricerca cliccando sul pulsante "Cerca". Al termine della ricerca, l'applicazione visualizza nella sezione "Risultati" un elenco di titoli e dei rispettivi autori. Cliccando su un titolo, l'utente puÃ² consultare la descrizione del libro nella sezione "Dettagli libro". Inoltre, tutte le ricerche effettuate vengono salvate e sono visibili nella sezione "Cronologia di ricerca".
